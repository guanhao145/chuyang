申请流程
Application Procedure
前期评估
初步了解学生背景，学生以及家长咨询申请建议，最终选择服务项目，并确定专业导师
深入了解
专业导师与学生深度沟通，了解学生申请意向，挖掘学生优势，参考美国大学权威数据库，量身打造申请规划
文书指导
明确申请学校文书要求，指导学生写作，帮助修改润色直到能够体现学生独特价值
套词指导
针对研究生申请，导师一对一指导套词、培训面试
网络申请
指导学生创建申请账号，提交申请材料，准时完成网申。导师跟进申请进度，并及时和学校沟通，确保所有申请材料准时提交
录取选校
学生拿到学校offer之后，帮助学生选择最终入读学校，并且申请 I-20表
后续服务
指导留美签证，疫苗体检，入校报道，以及申请宿舍或租房等
